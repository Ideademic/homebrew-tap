# Ideademic Tap

A [Homebrew](https://brew.sh) tap for Ideademic packages on macOS and Linux.

## Installation

### Add the tap, then install by name

```sh
brew tap ideademic/tap
brew install <formula>
```

### Or install directly without adding the tap first

```sh
brew install ideademic/tap/<formula>
```

### Brewfile

```ruby
tap "ideademic/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
